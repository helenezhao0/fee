# 保险手续费率计算器

一个基于HTML的保险手续费率计算工具，支持本地运行和手机端访问，数据来源于手续费统计表。

## 功能特点

### 1. 数据输入功能
- **险种代码**：6位数字输入，自动关联险种名称
- **险种名称**：支持模糊搜索，选择后自动填充险种代码
- **交费期间**：下拉选择，数据来源于手续费统计表C列
- **保险期间**：下拉选择，数据来源于手续费统计表D列
- **保单生效日期**：yyyymmdd格式输入，自动验证格式

### 2. 计算结果展示
- 新单手续费率(%)
- 续期一期手续费率(%)
- 续期二期手续费率(%)
- 续期三期手续费率(%)
- 续期四期手续费率(%)

### 3. 响应式设计
- 支持桌面端和移动端访问
- 自适应不同屏幕尺寸
- 优化移动端操作体验

## 使用方法

### 本地运行
1. 下载 `insurance_fee_calculator.html` 文件
2. 用任意浏览器打开该文件即可使用

### 部署到GitHub Pages
1. 将 `insurance_fee_calculator.html` 文件上传到GitHub仓库
2. 在仓库设置中开启GitHub Pages功能
3. 选择该HTML文件所在的分支和路径
4. 访问生成的GitHub Pages链接即可在手机端使用

## 数据来源
- 所有下拉菜单选项和手续费率数据均来源于 `手续费统计表.xlsx`
- 数据已预先处理并嵌入到HTML文件中，无需额外加载Excel文件
- 支持日期范围匹配，自动筛选生效期间内的手续费率

## 技术特点
- 纯前端实现，无需后端服务器
- 使用原生JavaScript，不依赖外部框架
- 本地数据处理，保护数据安全
- 实时表单验证，提供友好的错误提示
- 支持键盘导航操作，提升用户体验
